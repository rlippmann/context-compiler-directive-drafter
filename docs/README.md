# Docs

This directory is reserved for specifications, design notes, and contract
documents for `context-compiler-directive-drafter`.

Documentation in this repository should preserve the same philosophy as
`context-compiler`:

- explain user-visible behavior before architecture in general-purpose docs
- keep formal guarantees precise in specification documents
- treat behavior examples and tests as part of the contract when applicable

- placeholder API created
